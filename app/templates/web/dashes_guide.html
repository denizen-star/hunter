App Dash (Application Dashboard)

URL: /dashboard

Key Features

Application Cards

Each application is displayed as a card showing:

  • Company Name: With match score percentage displayed next to it (e.g., "85%")
  • Job Title: Position name with networking contact badge (if applicable)
  • Flag Button: Click to mark important applications for quick access via the "Flagged" tab
  • Achievement Badge: Icon reward for max accomplished networking badge
  • Metadata: Applied Date (ðŸ“…), Posted Date (ðŸ“‹), Last Updated Date (ðŸ”„)
  • Contact Count: Shows number of networking contacts (ðŸ‘¥) if you have contacts at this company
  • View Summary Button: Click to open detailed application summary page

Tabbed Interface

The dashboard organizes applications into status tabs:

  • Active (default) - All applications except rejected
  • Pending - Applications not yet submitted
  • Applied - Submitted applications
  • Contacted - Applications where you've contacted someone
  • Hiring Mgr - Applications where you contacted a hiring manager
  • Interviewed - Applications in interview process
  • Offered - Applications with job offers
  • Accepted - Accepted offers

Statistics Cards

Top of dashboard shows quick stats:

  • Total Applications
  • Pending Count
  • Applied Count
  • Interviewed Count
  • And more status-specific counts

Dynamic Counts: Each tab shows the number of applications in parentheses (e.g., "Applied (15)")

Advanced Sorting

  • Updated Timestamp (Newest/Oldest)
  • Applied Timestamp (Newest/Oldest)
  • Job Posted Date (Newest/Oldest)
  • Match Percentage (Highest/Lowest)

Network Dash (Networking Dashboard)

URL: /networking

Key Features

Contact Tiles

  • Person Name & Company
  • Job Title
  • Status Badge
  • Match Score
  • Timing Indicator (color-coded border)
  • Next Step suggestion
  • View Button

Status Filters

  • To Research
  • Contacted
  • In Conversation
  • Meetings
  • New Connection
  • Nurture
  • Referral Partner
  • Cold/Archive

Timing Filters

  • All
  • Recent (7 days, green)
  • Needs Activity (8-15 days, yellow)
  • Approaching (15-30 days, yellow)
  • Overdue (30-60 days, red)
  • Nurture (60-90 days, blue)
  • Cold (90+ days, gray)
