Tracking Guide

Reports Page

URL: /reports

When to Use: Check weekly or bi-weekly for overall pipeline status, identify applications needing follow-up, review status distributions, and see flagged items.

Key Features

Period Selector

Filter data by time period: Today, Yesterday, Last 7 Days, Last 30 Days, or All Time. Change periods to see trends over different timeframes.

Type Filter

View Jobs only, Networking contacts only, or Both combined. Combined view shows networking statuses with "Net:" prefix for clarity.

Status Distribution Charts

Visual pie charts showing how many applications/contacts are in each status. Quickly see where your pipeline is concentrated.

Activity Charts

Daily and cumulative activity charts showing application volume over time. Identify application velocity trends.

Follow-up Needed List

Automatically lists all items (jobs or contacts) that haven't been updated in 7+ days. Click links to go directly to items needing attention.

Flagged Items

Shows all flagged applications and networking contacts in one place. Great for managing high-priority opportunities.

Why Use Reports?

  • Quick Status Check: Get a high-level view of your entire pipeline in seconds
  • Identify Stale Items: The follow-up list highlights items that need attention
  • Track Progress: Activity charts show if you're maintaining consistent application volume
  • Manage Priorities: Flagged items section helps focus on high-value opportunities
  • Weekly Reviews: Perfect for weekly pipeline reviews to ensure nothing falls through cracks

How to Use Reports

  1. Select Time Period: Choose the timeframe you want to analyze (default: Last 30 Days)
  2. Choose Type: Select Jobs, Networking, or Both depending on what you want to review
  3. Review Charts: Look at status distribution and activity charts to understand pipeline composition
  4. Check Follow-ups: Review the "Follow-up Needed" list and click links to update stale items
  5. Review Flagged: Check flagged items section for priority opportunities
  6. Take Action: Use the information to prioritize your work for the week

Analytics Page

URL: /analytics

When to Use: Review monthly or quarterly for strategic insights. Use when you want to optimize your job search strategy, understand what's working, identify skill gaps, or analyze performance patterns.

Key Metrics & Insights

Response Rate

Percentage of applications that received company responses. Track this over time to see if your applications are improving.

Average Time to Response

Mean, median, min, and max days from application to first response. Helps set follow-up expectations.

Time to Response Distribution

Shows how many responses came in 0-1 days, 2-3 days, 4-7 days, etc. Understand typical response patterns.

Success by Match Score

Response and interview rates grouped by match score ranges (0-50%, 50-70%, 70-85%, 85-100%). Identify optimal match score targets.

Best Performing Day of Week

Which days you applied correlate with highest response rates. Optimize your application timing.

Application Velocity

Daily and cumulative charts showing application volume over time. Track consistency and growth.

Company Type Analysis

Response, interview, and offer rates by company type (startup, enterprise). Focus on where you perform best.

Location Insights

Response and interview rates by location. Identify geographic patterns in your search.

Salary Range Distribution

Distribution of applications across salary ranges. Understand where you're applying.

Skills Gap Analysis

Most requested skills, missing skills from your resume, and learning priorities. Identify skills to develop.

Skill Match Trends

How your match scores for top 10 skills have changed over time. Track skill relevance trends.

Why Use Analytics?

  • Optimize Strategy: Understand what's working and what isn't in your job search
  • Set Realistic Expectations: Know average response times to plan follow-ups
  • Identify Skill Gaps: See which skills appear frequently but you're missing
  • Focus Efforts: Concentrate on company types, locations, or match scores that perform best
  • Track Improvement: Monitor metrics over time to see if your applications are getting better
  • Learning Priorities: Get data-driven recommendations on which skills to learn next

How to Use Analytics

  1. Review Core Metrics: Start with Response Rate and Time to Response to understand baseline performance
  2. Analyze Match Scores: Check "Success by Match Score" to see if targeting higher matches improves results
  3. Check Timing: Review "Best Performing Day" to optimize when you submit applications
  4. Review Company Types: See if you perform better with startups or enterprises
  5. Identify Skill Gaps: Check "Skills Gap Analysis" to see top missing skills
  6. Set Learning Goals: Use "Learning Priorities" to focus skill development
  7. Track Trends: Review monthly to see how metrics change over time

Daily Activities Page

URL: /daily-activities

When to Use: Review daily or multiple times per week. Perfect for daily standups with yourself, understanding activity patterns, and ensuring consistent engagement with your pipeline.

Key Features

Chronological Timeline

All activities (job applications and networking contacts) sorted by date and time. See everything that happened in order.

Collapsible Day Sections

Activities grouped by day with collapsible sections. Click day headers to expand/collapse. Each day shows activity count.

Activity Type Icons

üíº Blue "Job" badge for job application activities. ü§ù Green "Networking" badge for networking contact activities.

Activity Details

Each activity shows: Company name, Position/title, Activity type (Created, Status Changed, etc.), Current status badge, Timestamp (EST).

Quick Navigation

Click on any activity to jump directly to the application or contact detail page for easy follow-up.

Activity Counts

Each day header shows the total number of activities. Quickly see which days were most active.

Why Use Daily Activities?

  • Daily Review: Perfect for daily check-ins to see what happened recently
  • Activity Tracking: Ensure you're maintaining consistent activity levels
  • Pattern Recognition: See if you have activity patterns (e.g., more active on certain days)
  • Nothing Missed: Chronological view ensures you don't miss any important updates
  • Quick Navigation: Easy way to jump to recent applications or contacts
  • Unified View: See both job and networking activities together in one timeline

How to Use Daily Activities

  1. Daily Check-in: Review each morning or evening to see recent activity
  2. Expand Today/Yesterday: Check the most recent days to see what's new
  3. Review Activity Types: See mix of job applications vs networking activities
  4. Click for Details: Click any activity to view the full application or contact
  5. Identify Gaps: Notice if certain days have no activity - plan to fill gaps
  6. Track Consistency: Ensure you're maintaining regular activity patterns

How These Tools Work Together

Daily: Check Daily Activities

Start each day by reviewing Daily Activities page to see what happened yesterday and what needs attention today.

Weekly: Review Reports

Once per week, check the Reports page to get overall pipeline status, review follow-up list, and check flagged items.

Monthly: Analyze with Analytics

Monthly review of Analytics page to understand performance trends, identify skill gaps, and optimize strategy for the next month.

Pro Tip: Use Daily Activities for tactical daily management, Reports for weekly pipeline reviews, and Analytics for strategic monthly optimization. Each tool serves a different purpose in your job search management.
