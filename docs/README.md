## üìã **Quick Reference**

### **Core Features**
- ‚úÖ **AI-Powered Matching**: Local AI (Ollama) analyzes job descriptions and resumes
- ‚úÖ **Automated Documents**: Generates cover letters, tailored resumes, and analyses
- ‚úÖ **Application Tracking**: Dashboard with status management and notes
- ‚úÖ **Daily Digest**: Automated daily summaries with status changes, activities, and analytics (MD/PDF/Email)
- ‚úÖ **Privacy-First**: 100% local processing, no cloud services
- ‚úÖ **Modern Interface**: Clean web UI with tabbed dashboard

### **Technology Stack**
- **Backend**: Python 3.11+ with Flask
- **AI**: Ollama (local LLM) with Llama 3
- **Frontend**: HTML, CSS, JavaScript (vanilla)
- **Storage**: File-based (YAML, Markdown, HTML)

### **System Requirements**
- Python 3.11+
- Ollama installed and running
- 8GB+ RAM available
- macOS/Linux/Windows
