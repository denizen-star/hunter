# Status Color Reference

This document lists all status colors used in Hunter for both Applications and Contacts.

## Application Status Colors

| Status | Background Color | Text Color | Description |
|--------|-----------------|------------|-------------|
| **pending** | `#fef3c7` (Yellow) | `#92400e` (Dark Brown) | Application created but not yet submitted |
| **applied** | `#dbeafe` (Light Blue) | `#1e40af` (Dark Blue) | Application submitted |
| **contacted someone** | `#f3f4f6` (Light Gray) | `#6b7280` (Gray) | Contacted someone at the company |
| **contacted hiring manager** | `#fee2e2` (Light Red) | `#991b1b` (Dark Red) | Contacted hiring manager |
| **company response** | `#eff6ff` (Very Light Blue) | `#1e40af` (Dark Blue) | Received response from company |
| **scheduled interview** | `#fef3c7` (Yellow) | `#92400e` (Dark Brown) | Interview scheduled |
| **interviewed** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Interview completed |
| **interview notes** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Interview notes added |
| **interview follow up** | `#fce7f3` (Light Pink) | `#9f1239` (Dark Pink) | Follow-up after interview |
| **offered** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Job offer received |
| **rejected** | `#fee2e2` (Light Red) | `#991b1b` (Dark Red) | Application rejected |
| **accepted** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Offer accepted |

| Status | Background Color | Text Color | Description |
|--------|-----------------|------------|-------------|

| **Sent LinkedIn Connection** | `#fef3c7` (Yellow) | `#92400e` (Dark Brown) | LinkedIn connection request sent |
| **Ready to Connect** | `#fef3c7` (Yellow) | `#92400e` (Dark Brown) | Ready to send connection |
| **Sent Email** | `#eff6ff` (Very Light Blue) | `#1e40af` (Dark Blue) | Email sent to contact |
| **Pending Reply** | `#eff6ff` (Very Light Blue) | `#1e40af` (Dark Blue) | Waiting for reply |
| **Connection Accepted** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | LinkedIn connection accepted |
| **Connected - Initial** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Initial connection established |
| **In Conversation** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Active conversation |
| **Meeting Scheduled** | `#fef3c7` (Yellow) | `#92400e` (Dark Brown) | Meeting scheduled |
| **Meeting Complete** | `#d1fae5` (Light Green) | `#065f46` (Dark Green) | Meeting completed |

## Color Categories

### Yellow/Amber (`#fef3c7` / `#92400e`)
- Pending actions
- Scheduled events
- Initial outreach

### Blue (`#dbeafe` / `#1e40af` or `#eff6ff` / `#1e40af`)
- Active applications
- Communication sent
- Research phase

### Green (`#d1fae5` / `#065f46`)
- Positive outcomes
- Completed actions
- Active relationships

### Red (`#fee2e2` / `#991b1b`)
- Rejections
- Negative outcomes

### Gray (`#f3f4f6` / `#6b7280`)
- Inactive states
- Neutral statuses

### Pink (`#fce7f3` / `#9f1239`)
- Follow-up actions
- Special attention needed
