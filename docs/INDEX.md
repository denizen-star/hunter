## üìñ What is Job Hunter?

Job Hunter is an AI-powered job application management system that:

1. **Extracts Features**: Analyzes your resume and job descriptions to extract technical skills, soft skills, experience, and certifications
2. **Calculates Match Scores**: Provides a 0-100% match score for each job
3. **Generates Documents**: Creates customized cover letters, tailored resumes, and detailed analyses
4. **Tracks Applications**: Maintains a dashboard of all your applications with status tracking

## üéØ Core Features

### Feature Extraction & Matching Engine
- **Enhanced Two-Phase System**: Preliminary matching (instant) + focused AI analysis (1-3 seconds)
- **3-5x Faster**: Reduced analysis time from 5-10 seconds to 1-3 seconds
- **More Accurate**: Match scores based on job requirements, not candidate skills
- **Detailed Analysis**: Categorized skills, unmatched skills, and targeted recommendations
- **60-80% AI Cost Reduction**: Preliminary matching reduces AI processing load

### Automated Document Generation
- Customized cover letters (integrates 50% of required soft skills)
- Tailored resumes optimized for each job
- Comprehensive qualifications analysis
- Beautiful summary HTML pages

### Application Management
- Clean dashboard interface
- Status tracking with notes
- Timeline of updates
- Easy document access

### Privacy & Cost
- 100% local AI processing with Ollama
- Zero API costs
- Complete privacy - data never leaves your computer
