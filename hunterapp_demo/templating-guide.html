Templating Guide

Learn how to create, manage, and use templates in Hunter to streamline your communication and maintain consistency across your job search outreach.

Related guide: How to Hunter? (see Phase I, Step 9.2 and Phase III)

What Are Templates?

Templates in Hunter are reusable message formats that help you maintain consistent, professional communication across your job search. You can create templates for different communication types (LinkedIn connections, emails, messages) and insert them into notes, cover letters, or outreach messages.

Key Benefit: Templates save time, ensure consistency, and help you maintain professional standards across all your communications. Create once, use many times!

Templates Page

URL: /templates.html

When to Use: Visit the Templates page to create new templates, edit existing ones, organize templates by delivery method, or delete templates you no longer need.

Key Features

Create New Templates

Use the "Create New Template" card to add templates:

  1. Select Delivery Method: Choose how the template will be used:
    • LinkedIn Connection
    • Email
    • LinkedIn Inmail
    • LinkedIn Message
    • Text
    • Phone
    • Other
  2. Enter Title: Give your template a descriptive name (e.g., "Recruiter Intro Email", "Thank You Follow-up")
  3. Write Content: Use the rich text editor to format your message with:
    • Headers, bold, italic, underline
    • Bulleted and numbered lists
    • Links and formatting
    • Colors and styling
  4. Save Template: Click "Save Template" to store it for future use

Saved Templates

Templates are organized by delivery method in collapsible groups:

  • Grouped Display: Templates grouped by delivery method (LinkedIn Connection, Email, etc.)
  • Collapsible Groups: Click delivery method headers to expand/collapse template groups
  • Template Counts: Each group shows how many templates it contains
  • Collapsible Templates: Click individual template titles to view/edit content
  • Delete Option: Each template has a delete button to remove unused templates

Why Use the Templates Page?

  • Centralized Management: All templates in one place for easy organization
  • Rich Text Editing: Format templates with professional styling
  • Delivery Method Organization: Group templates by use case for quick access
  • Template Library: Build a library of proven message formats
  • Consistency: Ensure all communications follow your professional standards

Where Can Templates Be Used?

Application Summary Pages

In the "Updates & Notes" tab of any job application summary page, you'll find an "Insert from Template" dropdown. Select a template to insert its content into the notes textbox for customization.

Networking Contact Pages

In the "Updates & Notes" tab of any networking contact summary page, use the "Insert from Template" dropdown to quickly add template content when documenting conversations or follow-ups.

Application Summary Pages

Location: Open any application from App Dash → Click "View Summary" → Go to "Updates & Notes" tab

How to Use Templates in Application Notes:

  1. Open Application Summary: Navigate to the application's detail page
  2. Go to Updates & Notes Tab: Click the "Updates & Notes" tab
  3. Find Template Dropdown: Look for "Insert from Template" dropdown above the notes textbox
  4. Select Template: Choose the appropriate template from the dropdown
  5. Template Inserts: Template content automatically populates the notes textbox
  6. Customize: Modify the template content with specific details (names, dates, company-specific info)
  7. Save: Add status if needed and click "Update" to save

Common Use Case: When updating an application status to "Contacted Someone", use the "Insert from Template" dropdown to quickly add a note about who you contacted and what you said, then customize it with specific details.

Networking Contact Pages

Location: Open any networking contact from Network Dash → Click contact tile → Go to "Updates & Notes" tab

How to Use Templates in Networking Notes:

  1. Open Contact Summary: Navigate to the networking contact's detail page
  2. Go to Updates & Notes Tab: Click the "Updates & Notes" tab
  3. Find Template Dropdown: Use "Insert from Template" dropdown in the notes section
  4. Select Template: Choose a template matching your communication type (LinkedIn, Email, etc.)
  5. Customize Content: Modify with contact-specific information
  6. Update Status: Change contact status if appropriate
  7. Save: Click "Update" to save the notes

Best Practices for Templates

Template Creation

  • Use Placeholders: Include placeholders like [Company Name], [Recruiter Name], [Position] that you'll replace when using the template
  • Keep It Generic: Write templates to be reusable across multiple situations
  • Include Key Points: Ensure templates cover all important information (who you are, why you're reaching out, what you want)
  • Professional Tone: Maintain consistent professional language across all templates
  • Appropriate Length: Consider platform limits (e.g., LinkedIn connection requests have 300-character limit)

Template Organization

  • Clear Naming: Use descriptive names that indicate purpose (e.g., "Recruiter Follow-up Email", "Thank You After Interview")
  • Group by Delivery: Assign appropriate delivery methods to organize templates logically
  • Regular Review: Periodically review and update templates to keep them current
  • Delete Unused: Remove templates you no longer use to keep the list manageable

Template Usage

  • Always Customize: Never send templates as-is. Always add specific details to personalize each message
  • Replace Placeholders: Make sure to replace all placeholder text with actual information
  • Review Before Sending: Always review customized templates before sending to ensure accuracy
  • Match Delivery Method: Use templates with delivery methods that match how you're actually communicating

Template Workflow

1. Create Template

Go to Templates page → Create New Template → Choose delivery method → Write content → Save

2. Use Template

Open Application or Contact → Updates & Notes tab → Select from "Insert from Template" dropdown

3. Customize

Template content appears in notes textbox → Replace placeholders → Add specific details → Personalize

4. Save & Use

Copy customized content → Use in actual communication (LinkedIn, Email, etc.) → Update status → Save notes

Common Template Types to Create

LinkedIn Connection Request

Template for initial LinkedIn connection requests. Keep under 300 characters. Include mutual interests or connection points.

Recruiter Intro Email

Professional introduction email to recruiters. Include your background, interest in the company, and position inquiry.

Hiring Manager Outreach

Direct outreach to hiring managers. More personal and specific than recruiter emails. Reference their work or company news.

Follow-up Email

Template for following up after initial contact. Thank them, reference previous conversation, and restate interest.

Thank You After Interview

Post-interview thank you message. Express gratitude, highlight key discussion points, and reinforce fit.

Status Update Note

Template for documenting status changes in Hunter. Include who contacted, what was discussed, and next steps.
